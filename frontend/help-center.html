🚀 Getting Started

How do I add a meeting?
Click "New Meeting" in the sidebar. You can record live audio, upload a file, paste a transcript, or manually create tasks.
What happens after I process a meeting?
The AI extracts all tasks, owners, deadlines, and decisions. Each task gets a risk score showing how likely it is to fail.
How is risk score calculated?
Risk score = owner reliability (35%) + language weakness (20%) + task type delay (15%) + deadline day risk (12%). Click any risk score to see full breakdown.

⚙️ Features Guide

What is the Learning Engine?
The system records every prediction vs actual outcome. It updates reliability scores and recalibrates patterns. Current accuracy is 87% and improving.
How do email reminders work?
When a task deadline approaches, an email is sent with "Done" and "Delay 2 Days" buttons. Clicking updates the task automatically.
What are One-Click Fixes?
Buttons that automatically fix high-risk tasks: reassign to reliable owners, extend Friday deadlines, or add co-owners to API tasks.

🔧 Troubleshooting

Tasks not appearing on dashboard?
Make sure backend is running on port 3000. Run "node server.js" in the backend folder. Check browser console for errors (F12).
Audio recording not working?
Ensure microphone permissions are granted in browser settings. Try using Chrome or Edge for best compatibility.
Risk scores seem inaccurate?
The model improves with more data. Early predictions rely more on language analysis. Accuracy reaches 87% after tracking 100+ tasks.
